# React / Vite / Axios / Tailwind / FSD

A react app that uses FSD architecture

To get started:
1. Fork the repo
2. Install all dependencies
    ```bash
    npm install
    ```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
vite or npm run dev
```

### Build the app for production

```bash
vite build
```
